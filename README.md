# Pewlett-Hackard-Analysis

## Overview of the analysis

This is an analysis to help prep the company Pewlett Hackard for their future “Silver Tsunami” of mass employee retirement. By determining the number of eligible retirees by job title and to identify which employees can participate in their new mentoring program. A program where eligible employees close to retirement can mentor new hires to help mitigate the “Silver Tsunami”. The company to start to plan their next step of action.
